THis is the blueprint for the "Everything-Remot" made by @Stock-Pot with improved ESP-Home Firmware.
The Blueprint offers collapsable section for each button and all button have single, double and long-press actions.

Since everybody want to customize their remote (i guess), i have added the name of the remote into the ESP-Home firmware
but to the Blueprint as well. Optional names in the section keep easier overview of the functions you did inside.
